REQUEST URL HTTP METHOD RESPONSE
Return All Movies "/movies" GET Returns an array of all JSON movie objects in database
Return single movie data "/movies/:title" GET Returns JSON object returning the data (description,director,genre etc.) of a single movie
EXAMPLE:
{ "title": "Ghostbusters",
"released": "1984",
"genre": [
"action",
"sci-fi"
],
"director": {
"name": "Ivan Reitman",
"biography": "Placeholder information",
"birthdate": "Placeholder date"
}
}
Return data on movie genres "/movies/genre/:genre" GET Returns information on genres via array of JSON movie objects
Return data of directors "/movies/:directorName" GET Returns JSON object containing information on director based on name
Register new user "/users" POST A JSON object adding the new user
Allow users to update their username "/users/:username" PUT returns data that was updated by the user
Allow users to add a movie to their favorites "/users/add/:movieName" POST Returns text that a movie was added successfully
Allow users to remove a movie from their favorites "/users/remove/:movieName" DELETE Returns text that a movie was removed successfully.
Allow users to delete their account "users/deleteAccount" DELETE Returns text that account was successfully deleted.
